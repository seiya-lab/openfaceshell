# OpenfaceExtract.py

`OpenfaceExtract.py` は、指定されたフォルダ内の動画ファイルを OpenFace を使用して処理し、出力フォルダに保存するスクリプトです。

## 前提条件

- OpenFace ツールがシステムにインストールされている必要があります。
- FFmpeg ツールがシステムにインストールされている必要があります。

## 使用方法

1. `OpenfaceExtract.py` スクリプトを実行します。
2. スクリプトの先頭にある以下の変数を適切に設定します。

   - `INPUT_FOLDER`: 入力フォルダのパスを指定します。処理する動画ファイルが含まれているフォルダです。
   - `OUTPUT_FOLDER`: 出力フォルダのパスを指定します。処理された結果のファイルが保存されるフォルダです。

3. スクリプトを実行すると、指定された入力フォルダ内の動画ファイルが順番に処理されます。
4. 処理中には、各動画ファイルの処理の進捗状況が表示されます。
5. 処理が完了すると、メッセージ "Processing completed." が表示されます。

## 注意事項

- 事前に OpenFace ツールと FFmpeg ツールをシステムにインストールしておく必要があります。
- 処理中は、OpenFace ツールと FFmpeg ツールが正しく機能することを確認してください。
- 入力フォルダ内の動画ファイルは、拡張子が ".mp4" のものとして扱われます。
- 処理された結果のファイルは、出力フォルダ内の個別のサブフォルダに保存されます。
- スクリプトを実行する前に、入力フォルダと出力フォルダが存在することを確認してください。
- スクリプトの実行には、適切な権限が必要です。
